# Steganography

- **steganography:** hiding messages
- ancient examples:
    - tattoo on messenger's head
    - blank wax tablets with messages inscribed on the wood beneath
    - invisible ink

## Null cipher
- message is in the last letters of each word
    - or some other position
- `Due in Havana tonight` -> `don't underestimate existing in no hurry - and v? a?, not a?. tonight, o`

## Bacon's cipher
- letters encoded in binary (5 digit groups)
- need 2 forms (e.g. capital + lowercase letters, 2 similar fonts, etc.)

## Image steganography
- round all pixel values to even numbers
- encode secret as odd-valued pixels
